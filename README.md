# operating_systems

Several exercises for Operating Systems (OS) Computer Science Course.

## Studied SysCall
- open
- read
- write
- close
- fork
- pipe and pipe2
- dup2
- getpid (and getppid)
- exec
- stat
- wait (waitpid)
- lseek
- usleep

## Signals
- signal
- kill
- pause

## Directories
- opendir
- readdir (also scandir)
